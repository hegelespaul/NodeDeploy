SuperMapeo

Este es un ejercicio académico desarrollado para una clase de la Universidad Autónoma Metropolitana (UAM) Lerma. En este proyecto, se demuestra el uso de Leaflet, Node.js y Android Studio para crear una aplicación móvil interactiva.


La app permite agregar marcadores en un mapa, cada uno con un título y una descripción, facilitando la visualización de ubicaciones personalizadas.


Todos los recursos empleados se pueden consultar en el repositorio de la clase.
